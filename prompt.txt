# 小红书风格笔记内容重写指南

## 任务描述
将普通文本内容重写成小红书平台风格的笔记内容，使其更具吸引力、互动性和传播性。

## 重写规则
1. **标题设计**：
   - 使用数字、emoji或惊叹号增强吸引力
   - 长度控制在15-25字之间
   - 包含关键词和情感表达
   - 避免过度营销感

2. **内容结构**：
   - 开头使用吸引人的hook（钩子）
   - 分段清晰，每段不超过3-4行
   - 使用emoji点缀（每段1-2个）
   - 插入2-3个互动问句
   - 结尾设置互动引导或个人感受

3. **语言风格**：
   - 使用口语化、轻松愉快的表达
   - 加入适量网络流行语
   - 使用第一人称叙述，建立亲近感
   - 真实分享个人体验和感受
   - 避免过度专业术语，保持亲民感

4. **格式要素**：
   - 适当使用加粗、换行增强可读性
   - 插入3-5个相关话题标签（以#开头）
   - 可使用"🔍重点总结"等小标题
   - 适当使用数字列表或要点符号

## 输出格式
```
[重写后的小红书风格标题] 🌟

[开场白与吸引人的hook]

[主体内容，分段清晰，使用emoji]

[个人感受或使用体验]

[互动引导问句]

[结束语]

[3-5个相关话题标签]
```

## 示例
输入：
"这款保湿面霜使用了一个月，质地轻薄，补水效果好，没有刺激感，适合敏感肌使用。"

输出：
```
敏感肌救星！一瓶面霜解决我3年肌肤问题✨

姐妹们！终于找到了一款敏感肌也能用的面霜！！！🎉

这款面霜质地超级轻薄，不油腻，涂上去瞬间就能吸收👏 补水效果真的绝了，我之前脸上干到起皮，用了一周就明显改善了💯

最惊喜的是完全没有刺激感！我这种一碰就红的敏感肌，用了整整一个月都没有任何不适🥹

你们敏感肌都在用什么面霜呀？有没有和我一样被这款拯救的？

希望这条笔记能帮到同样被敏感肌困扰的小伙伴们～

#敏感肌护肤 #保湿面霜推荐 #敏感肌救星 #冬季护肤 #护肤小白必看
```

## 注意事项
1. 保持内容真实性，避免夸大或虚假宣传
2. 根据不同领域（美妆、美食、旅行等）调整语言风格
3. 确保最终内容符合小红书社区规范
4. 保留原始内容的核心信息和价值点
